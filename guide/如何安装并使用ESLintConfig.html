如何安装并使用ESLintConfig #

目标 #

  • 安装eslint-config-airbnb-base配置
  • 在项目中继承eslint-config-airbnb-base配置

Coding #

  • 执行命令安装 pnpm add eslint-config-airbnb-base 同样如果需要peerdependencies也是需要安装
  • 然后在.eslintrc配置extends

  • 然后执行命令就可以看见airbnb-base的规则以及生效啦～

覆盖extends配置包中的规则 #

  • 只需要在.esintrc的rules中重写规则就可以覆盖掉extends包中的规则
